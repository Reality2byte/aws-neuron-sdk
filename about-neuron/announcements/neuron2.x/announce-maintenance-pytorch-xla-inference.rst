Torch/XLA support for Inference on Inf2 and Trn1 instances is now in maintenance mode starting with Neuron 2.31
------------------------------------------------------------------------------------------------------------------

Starting with :ref:`Neuron Release 2.31.0 <neuron-2.31.0-whatsnew>`, Torch/XLA support for inference on Inf2 and Trn1 instances is in maintenance mode. In maintenance mode, Torch/XLA support for inference on Inf2/Trn1 is based on PyTorch 2.9 and will address critical security issues and critical bug fixes only.
